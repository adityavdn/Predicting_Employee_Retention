# 👔 Predicting Employee Retention

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![HR Analytics](https://img.shields.io/badge/HR-Analytics-green.svg)

## 📖 Overview

Machine learning model to predict employee retention and attrition risk. Analyzes job satisfaction, performance, compensation, work-life balance, and career progression to identify flight risks.

## 🎯 Objectives

- Predict employee turnover
- Identify retention factors
- Workforce planning insights
- HR strategy optimization

## 🛠️ Tech Stack

- Scikit-learn, XGBoost
- Pandas, NumPy
- Matplotlib, Seaborn

## 🚀 Usage

```bash
git clone https://github.com/adityavdn/Predicting_Employee_Retention.git
pip install -r requirements.txt
jupyter notebook
```

## 📊 Key Factors

- Job Satisfaction
- Salary, Benefits
- Work-Life Balance
- Career Growth
- Manager Relationship
